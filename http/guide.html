üó≥Ô∏è/poll createüó≥Ô∏è

Create a prediction/poll

Usage : /poll create {type of poll} {question asked} {choice A} [choice B] [choice C] ...
Example : poll create |Make a Bet| |I will win this one !| |I don\'t think so| |Yes sure !|

Conditions : Type, Question and Choice A are required, alternatives choices are conditionnals up to 5.
(Prediction that lasts more than 1 week can be ended by a PollPy Admin if you don't have a patreon bot for your server !)

üîí/poll locküîí

Lock a prediction

Be Careful : Will lock the prediction and nobody will be able to interract with except with /poll end

üí∏/poll endüí∏
